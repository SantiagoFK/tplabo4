- node y express para el backend.
- nodemon para reiniciar server automaticamente ante los cambios.
- morgan para logging de requests.
- mongoose como ODM para interactuar con mongodb.
- dotenv para variables de entorno como url de bdd, y puerto.
- bcrypt para encriptar y desencriptar contraseñas de usuarios.
- validator para validar emails en el back.
- jsonwebtoken para manejar autorización con usuarios y jwt.
- socket.io para la implementacion de un chat por medio de sockets (servidor).
- socket.io-client para la manejar los sockets desde el cliente de Angular.
